# OdinSmash Extension

A powerful token analysis tool for Odin.fun that helps you assess token risks and make informed trading decisions.

## Features

- Real-time token risk analysis
- Developer behavior tracking
- Supply distribution metrics
- Integrated header indicator
- Token holder analysis

## Installation

### Quick Install (Chrome Web Store - Coming Soon)
1. Visit the Chrome Web Store page (link coming soon)
2. Click "Add to Chrome"
3. Confirm the installation

### Manual Installation
1. Download the [latest release](link-to-your-release.zip)
2. Unzip the file
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" in the top right
5. Click "Load unpacked"
6. Select the unzipped extension folder

## Usage

1. Visit any token page on [Odin.fun](https://odin.fun)
2. Look for the OdinSmash indicator in the header
3. View detailed analysis and risk assessment
4. Make informed trading decisions based on the data
